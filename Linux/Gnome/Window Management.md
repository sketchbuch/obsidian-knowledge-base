## Centre a Window

A keyboard shortcut to centre the current window with: <kbd>Super</kbd> + <kbd>Shift</kbd> + <kbd>c</kbd>

```bash
/org/gnome/desktop/wm/keybindings/move-to-center = ['<Super><Shift>c']
```
